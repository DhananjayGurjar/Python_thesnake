# 🐍 Python_thesnake

A  day-by-day Python practice repository documenting my journey through Python fundamentals — one concept at a time. Each `dayXX.py` file captures what I learned and experimented with that day, along with inline notes explaining the "why" behind the syntax.

## 📖 About

This repo is my personal log of learning Python from the ground up. Instead of just writing code, I add comments alongside each snippet to reinforce the concept and make it easy to revisit later. It also includes a small collection of LeetCode practice solutions as I start applying these fundamentals to problem-solving.

## 📂 Repository Structure

```
Python_thesnake/
├── day01.py        # print() basics — sep, end, and taking input
├── day02.py        # ...
├── day03.py        # ...
├── day04.py        # ...
├── day05.py        # ...
├── day06.py        # ...
├── day07.py        # Functions — simple & parameterized functions
├── swap_case.py     # Practice script: swapping character case in a string
└── leetcode/        # LeetCode problem-solving practice
```

## 🧠 Topics Covered So Far

- **Day 1** — `print()` statement basics: using `sep` and `end`, combining multiple values in one print call, and taking user input with `input()`
- **Day 7** — Functions: writing simple functions, passing arguments, and why functions make code reusable and efficient
- **swap_case.py** — String manipulation practice (case swapping)
- **leetcode/** — Problem-solving exercises to apply core concepts

*(Days 2–6 build on these fundamentals — variables, operators, conditionals, and loops — as part of the same step-by-step learning path.)*

## 🚀 Getting Started

Clone the repo and run any file directly with Python:

```bash
git clone https://github.com/DhananjayGurjar/Python_thesnake.git
cd Python_thesnake
python day01.py
```

**Requirements:** Python 3.x
